Arrays, Objects & Loops

Definitions / HW Review

Remember Variables?

A variable remembers a value that you've assigned to it. For example var x = 102.

Method

When an object contains a function that let's you interact with the object's properties.

Object

Essentially, everything in JavaScript is an object. You can create object when you need to store multiple variables to a particular place.

Property

A variable that provides information about an object. Needs to be paired with a value.

Array

A way of assigning multiple values to one variable.

Loop

Allows you to "loop through" each object in an array and perform a function on each.

Arrays

Examples

  • var x = [100, 454, 900, 128934, 2];
  • var fruits = ["orange","grape","apple","strawberry"];

Array Index

Each object in an array is assigned an index value, starting with 0.

  • 0 = orange
  • 1 = grape
  • 2 = apple
  • 3 = strawberry

Accessing an Item

To grab an item from an array, you simply need to refer to the variable holding the array, and then the item's index.
var myFruit = fruits[2];

Objects

Storage

When storing variables to an object, you need to specify both the property and its value.

Examples

  • weddingGuests = {
       name:"John Doe",
       age:"42",
       meal:"chicken"
    };
  • You can access any object property through the dot syntax: weddingGuests.name

Object Methods

An object can also contain a function that let's you interact with the object's properties. This is called a method.

Method Examples

myCountry = {
   getCapital: function() {
      alert(this.myCapital);
   },
   myName: "USA",
   myCapital: "Washington, D.C."
};


To call this method: myCountry.getCapital()

Everything is an Object

This is where it gets confusing - essentially everything in javascript and jQuery is an object.

Arrays, functions, elements, strings, etc. And every element has properties and methods.

Loops

Types

  • The for loop will let you repeat code a defined number of times
  • The do...while runs your code until a particular condition has been met

Loop Parts

Each loop has four parts that need to be defined

  • Initialization: Takes place once at the start of the loop
  • Test condition: Checks the variable to see if the loop should stop or keep going
  • Action: The code that is repeated as we go through the loop
  • Update: Updates the variable being used by the test condition to see if we should keep going

For Loops

myArr = ["one","two","three","four"];

for ( var i=0 ; i < myArr.length ; i++){
   alert("myArr[i]")
}

Lowercase i

In JS, the lowercase stands for "index" and represents the index of an object in an array. In loops, we use i to represent each index.

do...while Loops

myArr = ["one","two","three","four"];

var i=0;
while( i < 3) {
   alert(myArr[i]);
i++;
}

jQuery .each()

The jQuery each method operates similar to a for loop in that it lets you iterate over an arry of objects and perform functions on them

Using .each()

var boxes = $(".box");

$.each(boxes,function(index,box){
   alert("box");
});

The RETURN

Definition

A way of passing parameters from one function to another. Remember, in JavaScript we are mostly dealing with local variables, meaning variables are only available to the function in which they're defined. Using return allows us to make those variables to other parts of our code, which may need them to operate.

Example

function num(x, y){
   if(x !== y) { return false; }
   if(x < 5) { return 5; }
}

When we call this function, we will pass two numbers, and the function will evaluate it and return back an answer based on the numbers we've passed in.
