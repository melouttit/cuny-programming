What is AJAX?

  • Stands for Asynchronous Javascript And XML
  • The process allows you to exchange data with a server without having the user refresh the page
  • It is a combination of technologies allowing multiple things to happen at once
  • The javascript allows your application to interact with the DOM
  • XML is a way of storing information using HTML-like tags. However...
  • ...Data can be stored in multiple ways, not just XML. JSON is another format that is popular.

XML

  • Stands for eXtensible Markup Language
  • Data formatted in a way that it can be shared across an array of development tools
  • Like HTML, it is tag-based, and easy to use

XML Example

  <books>
    <book>
      <title>Lullaby</title>
      <author>Chuck Palahniuk</author>
    </book>
    <book>
      <title>House of Leaves</title>
      <author>Mark Danielewski</author>
    </book>
  </books>

JSON

  • Stands for JavaScript Object Notation
  • Like XML, formatted for use across tools and environments
  • Easier to generate, write and use than XML
  • Works best with Javascript, which is why it has become a preferred data structure

JSON Example

{
  "books":[
    {
      "title":"Lullaby",
      "author":"Chuck Palahnuik"
    },
    {
      "title":"House of Leaves",
      "author":"Mark Danielewski"
    }
  ]
}

Google Docs

One trick that has come in handy is the use of Google Docs to generate a dataset.

Why this is useful

  • Everyone already knows how to use Google Docs
  • We don't have to worry about hosting any data ourselves
  • Makes collaboration easier
  • Can serve as the CMS for a very basic site, eliminating the need for a large install, like WP

Getting Started

  • All you need is a Google Spreadsheet with some data in it
  • Here's one to get us started
  • In your Javascript you can do a regular JSON request, using your spreadsheet's url
  • Google URL structure: https://spreadsheets.google.com/feeds/key/worksheet/od6/public/values?alt=json-in-script&callback=?
  • The worksheet ID is found in the URL to your spreadsheet as the value for key
  • Make sure you Publish to Web under File in the menu.
