Note on Strings

In Javascript, strings can contain anything, including quote marks, which are also used to designated the beginning and end of a string. This can cause problems...

The Problem

var str = "Her favorite book is "The Thornbirds" and she was named after the main character.";

In this example, the string thinks you are ending after "is" and starting a new string at "and," which will make your variable undefined.

The Solution

There are several ways to get around this error

Use Single Quotes

var str = 'Her favorite book is "The Thornbirds" and she was named after the main character.';

Use Curly Quotes

var str = "Her favorite book is “The Thornbirds” and she was named after the main character.";

Use ASCII Codes

var str = 'Her favorite book is “The Thornbirds” and she was named after the main character.';

  • ASCII characters start with &# and end with ;
  • Opening quote: &# + 8220 + ;
  • Closing quote: &# + 8221 + ;

Escape the Quotes

var str = 'Her favorite book is \"The Thornbirds\" and she was named after the main character.';

What is a plugin?

  • A reusable script that programmers can "plug in" to their page to perform certain functions
  • The point is to reduce the amount of original code one has to write
  • They are especially useful for common features like photo galleries and countdowns
  • When using jQuery, make sure to search for plugins written using the jQuery framework
  • You can use the jQuery plugin registry to find plugins for your project (or just search Google)

Integrating a plugin

While each plugin usually comes with specific instructions, all (jQuery) plugins begin with the following steps:

  • Make sure you have jQuery included on your page
  • Download the plugin
  • Move the plugin files to a directory on your site
