Directory Structure

Most websites are organized by folders, or directories

  • The root folder is the most top-level folder
  • Each section of a site can be placed in a separate folder
  • The terminology is similar to that of a family tree with parent-child relationships
  • The main homepage of a site or section is index.html, which is returned when no other file type is specified in the URL
  • For example: http://mysite.com will return the same thing as http://mysite.com/index.html

The URL

  • Each image and file on the page has a URL under that site's domain
  • The URL path points to the particular file or directory you want retrieve
  • For example:
    • http://mysite.com/writing is the link to the Writing page
    • http://mysite.com/about.html is the link to the About page
    • http://mysite.com/images/artoo.jpg is the path to the artoo.jpg image

Relative URLs

When linking to a page on your own site, you can use shorthand

  • SAME: When linking to a file in the same folder, just use the file name
  • CHILD FOLDER: Use the name of the child folder + forward slash + file name
  • GRANDCHILD FOLDER: Name of the child folder + forward slash + name of grandchild folder + forward slash + file name
  • PARENT FOLDER: Use ../ to indicate the folder above the current one + the file name
  • GRANDPARENT FOLDER: Repeat ../ to go up two folders + the file name

The Target Attribute

If you want the link to open in another window or tab, you can set the target attribute to blank:

<a target="_blank" href="http://mysite.com">Go to My Site</a>

Linking an Email

If you want to link someone's email so that when the user clicks it their designated mail program opens up:

<a href="mailto:mymail@gmail.com">Email Me!</a>

Anchor & Deep Links

For linking to a specific part of a web page

  • First, identify the different points on your page using IDs:
    • The top of your page: 
      <h1 id="top">How to Code</h1>
    • Various sections: 
      <h2 id="css">What is CSS?</h2>
  • To then link to these points, either on your site or from another site:
    • Within your site: 
      <a href="#top">To the Top</a>
    • Linked from another site: 
      <a href="http://mysite.com/#css">Learn CSS from MySite.com!</a>

Meta Tags & SEO

  • Meta tags provide more information about your page to browers
  • They live in the head tag
  • Are useful for SEO purposes as search engines use them to grab more information about your site

Most Common Meta

  • Description: A description of the page and often shown in search results
  • Robots: Indicates whether search engines should add this page to their results.
  • Author: The author of the web page
  • Keywords: A list of words separated by commas that users might search to find your site*
  • You can also specify variables like time updated, a refresh interval, and prevent browers from caching the page, among other things

Meta Examples

  • Description: 
    <meta name="description" content="A website about Meghan Louttit
     and her programming and design work for The New York Times, The Washington Post"     />
  • Robot: 
    <meta name="robots" content="nofollow" />
  • Author: 
    <meta name="author" content="Meghan Elizabeth Louttit" />
  • Keywords: 
    <meta name="keywords" content="Meghan Louttit, portfolio, resume, design,
     web design, The New York Times, The Washington Post, multimedia"     />

More HTML

Structural Markup: The elements you can use to describe text like headings and paragraphs. The browser will interpret these to look a certain way.

Semantic Markup: The tag itself provides extra information to the browser but it does not look different to the user unless specified by a stylesheet.

Some Structural Tags

  • Superscript: 
    <p>Her birthday is the 4<sup>th</sup> day of June.<p>
  • Subscript: 
    <p>There is a lot of CO<sub>2</sub> in the atmosphere.<p>
  • Line Breaks: 
    <p>Wal-Mart found to have issued <br /> millions in bribes in Mexico City.<p>
  • Horizontal Rules: 
    <p>"The Thornbirds" by Colleen McCullough <hr /> "Eating Animals" by Jonathan Safran Foer<p>

Some Semantic Tags

  • Strong and Emphasis: 
    <strong>This is strong text!</strong>
    and 
    <em>This is emphasized text!</em>
  • Blockquote: 
    <blockquote>
     <p>Where we love is home. Home that our feet may leave but not our hearts.</p>
    </blockquote>
    <p>This is a quote from Olive Wendell Holmes. He said a bunch of other things, too.</p>
  • Insert & Delete: 
    <p>This class is going to start at <del>5:00</del> <ins>6:00</ins> p.m. on Mondays.</p>

  • There are several steps to putting an image in a web page, most happen before you do any coding

Download and Prep

  • Download the image or upload from your camera
  • Make sure to crop the image to the size you'll be using it on your page
  • Save it as one of the following file types:
    • .jpg is for images with many colors, typically photographs
    • .gif is for images with fewer and bolder colors, good for illustrations and graphics
    • .png transparent, also good for graphics
  • Save to the correct directory in your site (probably your "images" directory)

Add to Your Site

  • Use the image tag: 
    <img>
  • There are several attributes you can use in your image tag: Source, Alt, Title, Width and Height
  • Source must be declared: 
    <img src="images/artoo.jpg" />
  • Alt and Title are used to provide more information about the photo. The title will appear in a tooltip and alt is read by screen reader software for people with visual impairments.

Containers

  • Inline Elements: Always appear to continue on the same line as their neighboring elements. Examples include anchors, strong and emphasis, images and span.
  • Block Elements: Ones that always appear to start on a new line in your browser. Examples include header tags, paragraphs, lists and div.
  • The div and span tags are especially useful for grouping these elements together and controlling certain elements through CSS

Grouping Inline: span

  • Use it to contain a section of text that should look different from surrounding text
  • Hold a number of inline elements
  • Needs classes or IDs to be of use to the reader

Grouping Blocks: div

  • Use it to group various elements on your page together
  • Think of the div as a container - you might want it to hold your navigation elements, or all your header elements.
  • Like the span, it usually has a class or ID to make it relevent for positioning and styling

Class & ID

  • Attributes that can be applied to any HTML tag. Used to target elements through CSS.
  • Class: Use it to identify more than alike elements within a page 
    <p>There are <span class="important">no subways
     running today</span>, according to <em>The New York Times</em></p>
  • ID: Use it to identify a unique element on a page 
    <div id="navigation">
        <ul><li>Home</li><li>About Me</li></ul>
    </div>

HTML Comments

Use to make notes in an html file. Especially useful for noting where div tags are closed

Example: 

<div id="navigation">
    <ul><li>Home</li><li>About Me</li></ul>
</div><!--END OF NAVIGATION-->
