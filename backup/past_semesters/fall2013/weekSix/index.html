Javascript, A History

An 'Old' Language

  • Was created in 1995 by Brendan Eich
  • In 1996-1997 was taken to the ECMA Board to set standards, which could then be implemented by other browsers
  • The base for modern javascript was released in 1999 as ECMAScript 3
  • Originally, Microsoft had no intention of properly implementing JS into IE
  • In the mid-late aughts, while there was turmoil between major players, the open source and developer communities were revolutionizing what could be done with JS

Present Day

  • In 2005, partly in response to a paper released by Jesse James Garret, JS began its renaissance, spearheaded by open source communities
  • Out of this, libraries like jQuery, Prototype, MooTools and more were released
  • The latest rounds of innovation have produced developments like the Nodejs platform, HTML5 APIs, Moustache templates, and more

Mobile Implications

With the fast surge of mobile devices and Apple's decision to forego supporting Adobe's Flash - which had previously been the go-to technology for creating on-page interactions and animations, Javascript became even more important because it was already native to browsers on any device.

Javascript's Purpose

  • Dynamically add elements to a web page
  • Add animations based on user actions
  • Alert users to form actions
  • Add motions and transitions to text and photos
  • Load data from a server only when a user needs it

Javascript v. jQuery

  • Javascript is the language, jQuery is the library
  • jQuery is a toolbox - a set of predefined functions or shortcuts

Adding Javascript to Your Page

The following methods are true whether you are using native javascript, a library like jQuery, plugins, etc.

Internally

You can add the following at any point in your HTML file, however, it is best to add it to the bottom of the page, before your closing </body> tag

<script type="text/javascript">Your Script Here</script>

Externally

Almost the same as adding internal script, except you add a source attribute to the <script> tag, and nothing in between

<script type="text/javascript" src="js/main.js"></script>

Using jQuery

  • Since jQuery is a library, we have to download the set of predefined functions
  • Then we include the library on the page like any other script
  • Make sure the script tag that pulls in the jQuery library is on the page before any script to write that uses jQuery

Javascript Terms

Values

Javascript recognizes the following value types:

  • Numbers: 42, 1025, 3.14
  • Boolean (Logic): True / False
  • Strings: "Hello, World!"
  • null: A special keyword denoting a null value
  • undefined: A property that is undefined

Variables

Variables are symbolic names for values in your code

  • Javascript is case-sensitive
  • You can declare variables in two ways:
    • Using the keyword var. For example var x = 1025; or var x = "Hello, World!";
    • Simply declaring. For example x = 1025. This method creates a global variable; however, and in most cases shouldn't be used.

Scope

  • When you declare a variable outside of a function it is called a global variable because it is available to all functions in a document
  • When you declar a variable inside a function it is called a local variable because it is only available inside that function

Object-Oriented Programming

  • Objects: In programming, something that has a value, which can be a variable, function or data structure
  • Class: In OOP this is an object that is used to created instances of itself - it is a blueprint for building a specific object
  • Methods: Behaviors associated with a class that are executed when the program is run
  • Extra Reading about OOP

Native Javascript

To reference methods, Javascript uses document to refer to your HTML and can access all HTML elements through referencing document.

You might see...

  • document.write to add text
  • document.createElement to create an HTML element
  • document.appendChild to add an element to the page

The jQuery Wrapper

jQuery(document).ready(function(){     });

It's best to wrap any jQuery you write in the ready function. This means that all the code in between the curly brackets will run only when the page has loaded, ensuring you have access to all the HTML all on the page.

Syntax

Commonalities

  • Like CSS, most lines need to end with a semicolon
  • Also like HTML and CSS, white space doesn't matter. Use line breaks and tabs to keep your code readable and manageable.
  • Line breaks do matter in some cases - an accidental line break can break up a variable declaration, for example.
  • Use your text editor to help spot syntax errors

jQuery Specific

  • To define a jQuery function, you would write jQuery(); and put your elements or functions inside the parentheses
  • In shorthand jQuery(); = $();

Selectors:
jQuery vs. CSS

  • jQuery selects elements the same way CSS does
  • Remember, selectors can be tags, classes and IDs
  • CSS:  h1 { display:none; }
  • jQuery:  $('h1').hide();

Hover & Click

Hovering

The jQuery hover method allows you to define mouse hover actions, both in and out. When you run $.hover(); you need to define both these functions. The first function is on hover, the second function is when the mouse is done hovering.

Hover Example

For example:
$.hover(function(){
   // do something
},function(){
   //undo that action
});

The Click

The jQuery click method is a very useful tool. When you run $.click(function(){ }); it runs the code inside your curly brackets on a click event.

Click Example

For example:
$.click(function(){
   alert("You've Clicked Me!")
});
