External Stylesheets

There are two ways to include a stylesheet on your HTML page.The way you choose depends on how you organize your styles. Depending on the size of your site and how separate you want to keep things.

  • link - In your HTML you can include a separate link tag for each stylesheet
  • import - You can include one stylesheet in your HTML and import separate stylesheets from there

Link

  • Create a file with the .css file type and save in your css directory
  • In your <head> element, include a <link> element for each stylesheet
  • Attributes of your <link> element are: href, rel and type
  • It should look like this: >link rel="stylesheet" type="text/css" href="style.css" />

Import

  • Create a file with the .css file type and save in your css directory
  • In your .css file, include @import statements before any other styles
  • It should look like this: @import url("style.css");

List Styles

Used on the list items in unordered and ordered lists using the property name: list-style

Unordered Lists

  • list-style: disc; (This is the default on most browsers)
  • list-style: circle;
  • list-style: square;
  • list-style: none;

Ordered Lists

  1. list-style: decimal;
  2. list-style: decimal-leading-zero
  3. list-style: lower-alpha;
  4. list-style: upper-alpha;
  5. list-style: lower-roman;
  6. list-style: upper-roman;

List Images

If you want to replace any of the specified bullets with an image of your choosing you can use the list-style-image property. Note that this can only be used on <ul> and <li> elements

  • list-style-image: url("images/icon.png");
  • As with any image, you can use external and interal link sources

Positioning

Through CSS you can control where an element sits on a page. Positioning is actually very powerful and comes in handy when you start thinking about screen sizes, how your site will look on various devices and responsive layouts.

  • Block Level elements start on a new line and act as the main buliding blocks for any layout
  • Inline elements flow between surrouding elements and text
  • You can control how much space an element takes up by adjusting height and width and to separate boxes you can use borders, margins, padding and background colors and images.

CSS Schemes

You can use the position property to specify an element's positioning scheme. The following options are available:

  • Normal flow means every block element appears on a new line. This is default browser behavior.
  • Relative positioning moves an element from the position it would be in a normal flow, moving it relative to where it would have been placed
  • Absolute positioning takes the element out of it's normal flow, meaning it does affect the position of surrounding elements
  • Fixed positioning is a form of absolute positioning, in that the element is removed from the normal flow, but then they are additional fixed on the page, meaning they don't move when the user scrolls

Z-Index

Use the z-index propety to control which element goes on top. To do this, both elements must have a z-index set

Clear

This property allows you to say that no element (within the same containing element) should touch the left or right sides of a box.

  • clear:both; clears the elements on both sides of a box
  • clear:left; clears elments on the left-hand side of a box
  • clear:right; clears elements on the right-hand side of a box
  • clear:none; elements can touch either side. this is the default
  • Applying the clear property to an empty <div> can help create space: <div style="clear:both;"></div>

Background Images

You can use background images on your page and use several different properties to control them

  • Set your background image: background-image:url("image.gif");
  • Decide whether it repeats by setting background-repeat to repeat, repeat-x, repeat-y or no-repeat
  • Decide whether the image scrolls or is fixed in the same place by settting backgroun-position to fixed or scroll
  • You can set the background-position with a pair of values representing the horizontal and vertical positions
