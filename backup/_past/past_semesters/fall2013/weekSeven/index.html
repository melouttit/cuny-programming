Interpreting the Console

Some common errors and what they mean

  • Failed to load resource means an external file can't be found. Make sure the file both exists in the directory you're pointing to and you are spelling the file name correctly
  • Unexpected identifier means your syntax is off and something - usually a semi-color, is in the wrong place and/or out of order.
  • Uncaught SyntaxError: Unexpected token means your opening and closing parentheses or brackets are out of order

Functions

  • The Syntax: function(){ };
  • Named Function: function readLips(){ };
  • Diagraming a function: What does each piece do

$()

The Wrapper - The dollar sign calls the jQuery library and indicates that a jQuery method will be used on the element in the wrapper.

document

The Selector - the element that you are going to run your method on. This can be the page itself (document) or a tag ('h1'), an ID ('#idname') or class ('.classname')

.ready()

The Method - whatever jQuery method you are running on the element you've specified. In this case we are running ready, which means to wait until the page is loaded before running the code.

function(){ }

The beginning of your function. The parenthese after function is where you can pass in other information that your function needs, but not neessary. The code you are going to run goes inside the curly brackets.

Animations

The .animate() method allows you to create custom animations

  • $(selector) is the element to be animated
  • .animate() calls the animate method
  • .animate({params},speed,callback);

Animation Resources

  • jQuery site
  • w3Schools Try It Yourself!

The Parameters

  • params defines the CSS properties to be animated. Not all CSS properties can be used. Must be expressed as a series of key: value pairs and separated by commas if more than one.
  • speed is the optional definiton for how fast to complete the animations. You can use the strings 'slow' and 'fast' or numerical values.
  • callback is a function that runs once the animation is complete. This is also optional

Display

There are several methods that allow you to hide an element. They set the inline style of your element to display:none; but does not change this in the stylesheet.

  • .animate() You can hide by setting height to 0 or toggle it
  • fadeOut() and fadeIn()
  • slideUp() and slideDown()
  • hide() and show()

Chaining

This is simply the term for executing several methods on one element. For example: $(selector).animate({left:'50px'},'500').fadeOut();

You can also use the delay() method to queue your animations:
$(selector).animate({
   left:'50px'
},'500').delay(500).fadeOut();
