List Styles

Used on the list items in unordered and ordered lists using the property name: list-style

Unordered Lists

  • list-style: disc; (This is the default on most browsers)
  • list-style: circle;
  • list-style: square;
  • list-style: none;

Ordered Lists

  1. list-style: decimal;
  2. list-style: decimal-leading-zero
  3. list-style: lower-alpha;
  4. list-style: upper-alpha;
  5. list-style: lower-roman;
  6. list-style: upper-roman;

List Images

If you want to replace any of the specified bullets with an image of your choosing you can use the list-style-image property. Note that this can only be used on <ul> and <li> elements

  • list-style-image: url("images/icon.png");
  • As with any image, you can use external and interal link sources

Positioning

Through CSS you can control where an element sits on a page. Positioning is actually very powerful and comes in handy when you start thinking about screen sizes, how your site will look on various devices and responsive layouts.

  1. Set the value of your position
  2. Then you can set where it sits based on the properties top, right, bottom and left

The Position Property

  • Normal flow means every block element appears on a new line. This is default browser behavior.
  • Relative positioning moves an element from the position it would be in a normal flow, moving it relative to where it would have been placed
  • Absolute positioning takes the element out of it's normal flow, meaning it does affect the position of surrounding elements
  • Fixed positioning is a form of absolute positioning, in that the element is removed from the normal flow, but then they are additional fixed on the page, meaning they don't move when the user scrolls

    • Z-Index

    Use the z-index propety to control which element goes on top. To do this, both elements must have a z-index set

    z-index: 100px;

    Clear

    This property allows you to clear elements from one or both sides of a box

    • clear:both; clears the elements on both sides of a box
    • clear:left; clears elments on the left-hand side of a box
    • clear:right; clears elements on the right-hand side of a box
    • clear:none; elements can touch either side. this is the default

    Background Images

    • Set your background image:
      background-image:url("image.gif");
  • You can decide how it repeats background-repeat
    • repeat
    • repeat-x
    • repeat-y
    • no-repeat
  • Decide whether the image scrolls or is fixed by settting background-position to fixed or scroll
  • You can set the background-position with a pair of values representing the horizontal and vertical positions
  • Ex: background:url('images/foo.jpg') no-repeat 5px 5px;
