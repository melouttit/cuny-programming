Programming for Journalists

An introduction to the three most used languages in front-end development - HTML, CSS and JavaScript - that connects these technologies to how they are used in modern newsrooms to do journalism.

You will NOT

  • Become a programmer (yet)
  • Become a designer (yet)
  • Become a hacker (yet?)
  • Use interface tools to build anything

You WILL Learn...

  • How the Internet works
  • How to build a website from scratch
  • How to use Git to develop and host a project
  • Basic design principles
  • To write basic HTML, CSS and JS from scratch
  • How these technologies work together
  • How to use your computer's CLI

The Goal

This course is an attempt to not only introduce you to programming, but to strip away the mystery behind how the Internet works, how files are created, how websites are stored and served, and to introduce you to common newsroom workflows for digital projects. We will use tools and systems that you will come across in most newsrooms today.

The goal is to make you familiar with the underlying processes so that as these tools inevitably evolve and change over time, you will have the base understanding and ability to troubleshoot that will allow you to adapt to these changes.
