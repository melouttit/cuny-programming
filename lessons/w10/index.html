jQuery Plugins

  • A common set of jQuery functions that can be added to your site for quick functionality
  • Anyone can write a plugin and release it for widespread use. Most will include documentation on how to implement and what custimzation is available to you
  • The jQuery plugin registry is a good starting point for seeing what is available.

Adding a Plugin

  • Download the plugin code from the source. On the homepage for your plugin, look for a download link.
  • Usually plugins will come with at least one JavaScript file, a CSS file and sometimes an images directory and README file.
  • Read the instructions for installing carefully. For most plugins, you will just include the source JS and CSS files on the page you want to instantiate the plugin on.
  • Make sure the jQuery library is on the page before the plugin

Example

  • Using Tiny Carousel
  • Click the Download button on the top right. Open the folder and you will see two options for responsive or simple.
  • In whichever you choose, you will JS and CSS files, an images directory and an index file.

Minified JS

You might see two JS files in the plugin you download with filenames that are similar except for a -min at the end. This means the owner is providing you with two ways to include their JS. The file with the -min at the end is the minified version, meaning it's stripped of white space and lighter to download. You would only want the unminified version if you were planning on altering the plugin. Open both files to see the difference.

Documentation

  • Look either in the plugins README file or on the website to find the documentation for how to use the plugin.
  • Most plugins require a standard HTML setup and then only a few lines of JS to get them working.
  • EXAMPLE: $(document).ready(function() {
      $('#slider1').tinycarousel();
    });
  • Where #slider is the HTML element you want to instantiate the plugin on and tinycarousel() is the plugin method that runs the code you need.

Options

  • Beyond the standard setup, most plugins have options you can pass in to the customize the plugin.
  • You will need to read through the plugins documentation to find the options that are available to you.
  • The plugin owner might also provide several stylesheets for you to choose from so you can customize the look of the plugin.

Option Example

  • EXAMPLE: $("#slider3").tinycarousel({
      bullets: true,
      buttons: false
    });
  • Where bullets: true is an option the plugin provides, allowing you to create bullets under your carousel. And buttons: false allows you to hide the previous/next buttons.

Demos

Many plugins also provide demo pages or have demos working on their page. Use the inspector to see how they set up the demos as a guide to setting up your own.
